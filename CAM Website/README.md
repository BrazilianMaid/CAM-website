# Channel Revenue Engine — Stage 2

This version of the site is organized into separate files so it is easier to maintain and improve.

## Structure

- `index.html` → page content and structure
- `css/styles.css` → styling and layout
- `js/script.js` → interactions and behavior
- `assets/` → future images, logos, icons, downloads

## How to use

Open `index.html` in a browser, or deploy the whole folder to Vercel or GitHub.

## Editing rule

- Change copy in `index.html`
- Change design in `css/styles.css`
- Change behavior in `js/script.js`
